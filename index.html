Welcome to the VLE-7 DevOps Playground

A lightweight demo that builds a Docker container, deploys it to Kubernetes, and is designed to be monitored with Prometheus and Grafana. This page demonstrates the complete development-to-production pipeline for a web application.

Containerized Web App

The app runs in an Nginx container built from a simple Dockerfile. It serves a polished static homepage and is ready for Kubernetes deployment.

Automated CI/CD

A Jenkins pipeline checks out the repository, builds the Docker image, loads it into Minikube, and applies the Kubernetes deployment manifest.

Cluster Monitoring

Prometheus alert rules are included for pod instability and high CPU usage, making the project suitable for observability demos and troubleshooting exercises.
